# Архітектура Monitor-AI

## Як працює система
Комп'ютер
↓ (psutil збирає дані)
Python Flask сервер (app.py)
↓ (відправляє кожну секунду)
JavaScript в браузері
↓ (малює графіки)
Користувач бачить дані в реальному часі
## Технічний стек
- Python 3.14.2 + Flask + psutil
- HTML5 + CSS3 + JavaScript
- Chart.js для графіків

## Структура файлів
- app.py — збирає дані і запускає сервер
- templates/index.html — що бачить користувач
- static/css/style.css — дизайн
- static/js/main.js — графіки і оновлення даних

## Що збираємо
- CPU: загальний % + кожне ядро окремо
- RAM: використано МБ / всього МБ / відсоток
- Диск: використано ГБ / всього ГБ
- Мережа: швидкість завантаження і відвантаження
- Оновлення: кожну секунду

## Дизайн рішення
- Темна тема: фон #0a0a0a
- Акцент: зелений #00ff88
- Карточки: #1a1a1a
- Шрифт цифр: монопросторовий

## Чому такий стек
- Flask — простий для початківця, легко зрозуміти
- psutil — стандартна бібліотека для системних даних
- Chart.js — не потрібно багато налаштувань
- Без Docker — навчальний проект, не продакшн
